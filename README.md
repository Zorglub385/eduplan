# EduPlan ✦ Branche de démo

> **Vous regardez la branche `Example`** — elle sert uniquement à alimenter la démo GitHub Pages avec un emploi du temps fictif anonymisé.  
> Pour utiliser EduPlan avec vos vrais cours → **[aller sur la branche `main`](https://github.com/Zorglub385/eduplan)**

---

## C'est quoi cette branche ?

GitHub Pages ne peut servir qu'une seule branche. Plutôt que d'exposer des données personnelles sur `main`, cette branche `Example` contient :

- **`config.js`** — configuré avec un URL pointant vers un fichier `.ics` local fictif
- **`calendrier_anonymise.ics`** — un vrai fichier iCal avec des cours inventés, des profs fictifs, des salles génériques
- **`index.html`** — identique à `main`, aucune modification

Le résultat : [zorglub385.github.io/eduplan/](https://zorglub385.github.io/eduplan/) affiche un emploi du temps complet et fonctionnel, sans aucune donnée réelle.

## Utiliser EduPlan avec vos propres cours

**→ [Aller sur la branche `main`](https://github.com/Zorglub385/eduplan)**

Puis :
1. [Télécharger le ZIP](https://github.com/Zorglub385/eduplan/releases/latest)
2. Ouvrir `config.js` → coller votre URL iCal Pronote
3. Double-cliquer sur `index.html`
