# Contribuer à EduPlan

Merci de vouloir contribuer !

## Signaler un bug
Ouvrez une Issue avec :
- Description claire
- Navigateur utilisé
- Étapes pour reproduire
- Capture d'écran si possible

## Proposer une amélioration
Ouvrez d'abord une Issue pour en discuter.

## Pull Requests
- Code propre et commenté
- Pas d'URL iCal personnelle
- Expliquez clairement les modifications
